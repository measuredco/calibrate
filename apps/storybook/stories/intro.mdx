# Components

Use this Storybook to explore the Calibrate component catalog.

Stories use the HTML / SSR renderers from `@measured/calibrate-core`.

`@measured/calibrate-react` exposes the same component props and emits the same markup.

- **Control** — button, checkbox, expander, fieldset, input, menu, radios, range, switch, textarea
- **Environment** — root, surface
- **Graphic** — avatar, icon, image, logo, pattern, shape
- **Layout** — box, container, divider, grid, inline, stack
- **Navigation** — link, nav
- **Status** — alert, badge, banner, spinner
- **Structure** — card, details, figure, page, panel, poster, sidebar
- **Typographic** — blockquote, heading, prose, text
