# Measured brand

## Visual language components

Note these SVGs are normally color with the brand azure or neutral palettes (via currentColor, etc.)

### The visual language

The core component is a rounded corner square and is the basis for the visual language system. It is the foundation for the corner element used in the brand identity and has multiple applications. Direct usage of the rounded corner square is restricted.

### The corner

The corner is derived from the rounded corner square and is a fundamental aspect of the Measured identity. It is the core element that is used in the logo and has multiple applications across the visual language system.

### The large tile slice

All components in the visual language are extracted from the corner or rounded corner square. All proportions must be maintained and fixed to create a unified and coherent visual system.
